## What is DocsPage?

DocsPage is the default documentation template that Storybook uses to generate documentation pages for your stories. When you navigate to the Docs tab of a story, Storybook renders a DocsPage that is composed of a collection of [Doc Blocks](./doc-blocks.mdx). Both the DocsPage itself and the individual blocks within it can be overridden to suit your needs.

The default DocsPage includes the following blocks:

- **[Title](../api/doc-blocks/doc-block-title.mdx)** — The story's name
- **[Subtitle](../api/doc-blocks/doc-block-subtitle.mdx)** — The component's description
- **[Description](../api/doc-blocks/doc-block-description.mdx)** — A JSDoc comment or the story's description
- **[Primary](../api/doc-blocks/doc-block-primary.mdx)** — The primary story rendered
- **[Controls](../api/doc-blocks/doc-block-controls.mdx)** — An interactive controls table
- **[Stories](../api/doc-blocks/doc-block-stories.mdx)** — All related stories
