Sets a list of directories of [static files](../../configure/integration/images-and-assets.mdx#serving-static-files-via-storybook-configuration) to be loaded by Storybook.

<CodeSnippets path="main-config-static-dirs.md" />

<Callout variant="info">

When using Vite-based frameworks, Storybook disables Vite's [public directory handling](https://vite.dev/guide/assets#the-public-directory) (`publicDir: false`) so that it doesn't copy assets that conflict with the directories listed above. If you rely on Vite's public directory, add it to `staticDirs` or re-enable it through [`viteFinal`](./main-config-vite-final.mdx).

</Callout>
